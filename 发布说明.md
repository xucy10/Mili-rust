# Mili-rust 服务器 - 构建和分发指南

## 快速构建

### Windows

```batch
# 双击 构建.bat 选择构建方式
构建.bat

# 或者直接运行
cargo build --release
```

### Linux/Mac

```bash
# 构建
cargo build --release

# 运行
./target/release/mili-rust
```

## 构建产物

构建成功后，可执行文件位于：

```
target/release/mili-rust.exe    # Windows
target/release/mili-rust        # Linux/Mac
```

## 分发给其他人

### 方式一：直接发送可执行文件

1. 运行 `cargo build --release`
2. 找到 `target/release/mili-rust.exe`
3. 发送给其他人
4. 对方双击运行即可（不需要安装Rust）

### 方式二：打包分发

```bash
# 创建发布包
mkdir mili-rust-server
copy target\release\mili-rust.exe mili-rust-server\
copy README.md mili-rust-server\
```

## 运行要求

### Windows
- Windows 10 或更高版本
- 无需安装其他依赖

### Linux
- glibc 2.17 或更高版本
- 无需安装其他依赖

### Mac
- macOS 10.15 或更高版本
- 无需安装其他依赖

## 性能对比

| 构建类型 | 大小 | 启动时间 | 运行性能 |
|---------|------|---------|---------|
| Debug | ~50MB | 快 | 慢 |
| Release | ~20MB | 中等 | 快 |

**推荐使用 Release 版本**

## 命令行参数

```bash
# 默认运行（端口25565）
mili-rust.exe

# 指定端口（需要修改代码）
mili-rust.exe --port 25566
```

## 常见问题

### 编译失败

确保安装了：
- Rust (https://rustup.rs)
- Visual Studio Build Tools (C++ 桌面开发)

### 运行闪退

检查端口是否被占用：
```batch
netstat -ano | findstr :25565
```

### 内存不足

调整系统虚拟内存或减少加载的区块数量

## 技术规格

- **协议版本**: Minecraft 1.20.1 (Protocol 763)
- **默认端口**: 25565
- **最大玩家数**: 受限于服务器性能
- **世界大小**: 无限（Anvil格式）

## 开发者

如需修改服务器逻辑，编辑 `src/main.rs` 后重新构建。
